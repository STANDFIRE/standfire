Individual selection

 Capsis Help

Principle

Select the elements with their identifiers: select the ids in the left side list and add them to the right side list. Individuals with these ids will be selected.

Selection of the elements

Click on an element in the list to select it.

Then, optionnaly:

  • click another element while pressing the Shift key to select a group of contiguous elements,

  • click other elements in the list while pressing the Control key to select elements which are not contiguous.

Move from one list to the other

Add the left side selected elements to the right side list
Remove the elements from the right side list
Add all the elements of the left side list to the right side list
Remove all the elements from the right side list
Sort both lists in ascending order
