True/false Filter

 Capsis Help

Principle

Activate and set to true or false the properties you choose to select the elements with the same values for these properties.

Proposed properties

The list of proposals contains the boolean properties (true or false) which are common to all the elements of the collection to be filtered.

It is possible for the collection to contain elements of several different types, for example several types of tree descriptions with a common part and some particularities. Only the common boolean properties are considered by this filter.

Only the activated properties are taken into account. To activate a property, check the first checkbox of the line: the name of the property becomes enabled. The second checkbox is used to set the wanted value for the property: true (checked) or false (unchecked).

If several properties are activated, the filter checks all the properties by processing a logical AND: the kept elements will match all the properties.
