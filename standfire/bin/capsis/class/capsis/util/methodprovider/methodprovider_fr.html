Méthodes de calcul
Aide Capsis

Ceci est la liste des méthodes de calcul ("fournisseurs de méthodes") disponibles dans Capsis. Les modules Capsis (ex: Samsara, Sylvestris, PP3...) peuvent implémenter certaines d'entres elles pour fournir la variable correspondante. Ces méthodes peuvent être auto-détectées par certains outils ou certaines extensions (extracteurs graphiques...) et utilisées pour calculer la variable correspondante pour différentes étapes d'un projet (histogrammes, courbes, tables...).

Pour chaque méthode de calcul, les modules proposant une implémentation ainsi que les extensions s'appuyant sur la méthode sont listés. Les sources sont consultables en cliquant sur le nom du module ou de l'extension. On peut ainsi par exemple vérifier comment un module calcule une grandeur ou quelles extensions s'appuient sur une méthode de calcul.

Il est possible d'enrichir les commentaires des méthodes dont vous êtes auteurs (uniquement) depuis cette page. Pour cela, votre navigateur doit être paramétré pour ouvrir votre éditeur (TextPad, SciTE...) quand vous cliquez sur les liens vers les codes sources. Modifiez les commentaires (ajoutez les unités, détaillez les explications insuffisantes...) puis sauvegardez (et recompilez si vous voulez vous assurer de ne pas avoir fait d'erreurs) et enfin, reportez les sources modifiés vers le serveur SVN (commit). Pour forcer la reconstruction de cette page à la prochaine consultation, supprimez la page du répertoire capsis4/bin/capsis/util/methodprovider/.
