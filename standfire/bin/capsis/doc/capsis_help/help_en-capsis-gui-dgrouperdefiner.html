Group Manager

Principle

Create groups (trees, cells…) to check them separately in the various viewers and data extractors.

Group tab

Identification

  • New group: specify its name
  • Existing group: modify the group name if needed

Elements type

  • Choose the type of the group elements (trees, cells…)

Group type

  • A static group retains only the ids of the elements selected at group creation time
  • A dynamic group retains the rules defined at group creation time and rerun them at each invocation

Definition tab

Use the filter manager to build your group.
