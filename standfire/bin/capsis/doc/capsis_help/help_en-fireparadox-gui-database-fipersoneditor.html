TEAM PERSONS EDITION

This interface aims at create, update, desactivate orreactivate persons inside a team.

PERSON CREATION

Enter the new person name, this field is compulsary.

  • Save in the database : Save the new person in the database
  • Cancel : Cancel and close window
PERSON UPDATE

Enter the person new name if necessary, this field is compulsary.

  • Desactivate/Reactivate : To desactivate or reactivate this person in the database.
  • Save in the database : Save the new person in the database
  • Cancel : Cancel and close window
PERSON DESACTIVATION

The person name appears on the screen for verification.
