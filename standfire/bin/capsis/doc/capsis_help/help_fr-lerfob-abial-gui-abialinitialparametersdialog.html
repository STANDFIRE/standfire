Création d'un projet AbiAl

Localisation

Indiquer le massif (Vosges ou Jura) et l'altitude de la station.

Initialisation du peuplement

Indiquer l'âge à la souche auquel la simulation doit commencer. On considère que l'âge à la souche est supérieur de 15 ans à l'âge compté à 1m30.

  • Générer un peuplement virtuel : pour générer un peuplement avec un rdi initial de 1 à 20 ans, indiquer soit la surface terrière, soit la densité (nombre d'arbres /ha) initiale et cocher l'une des deux cases 'Auto'. Pour générer un peuplement avec un rdi quelconque, décocher les 2 cases 'Auto' et donner la surface terrière et la densité initiales.
  • Charger un fichier d'inventaire : cliquer sur le bouton Charger pour choisir un fichier et mettre à jour la description du peuplement avec les informations contenues dans l'entête du fichier. Si vous modifiez ensuite ces données (massif, altitude…) les données du fichier ne seront pas prises en compte.
Classe de fertilité

Préciser la classe de fertilité en donnant une hauteur dominante à un âge à la souche donné. On considère que l'âge à la souche est supérieur de 15 ans à l'âge compté à 1m30.

Paramètres de simulation

La surface (1 ha par défaut) joue directement sur le nombre d'arbres qui seront générés. Réduire la surface permet donc de réduire le temps de simulation et la mémoire requise. Les sorties graphiques de Capsis comportent en général une option “Par hectare” dans le menu de configuration (clic droit sur le graphique) qui permet de ramener les résultats (volumes, surfaces..) à une surface de 1 ha.

Le pas de temps (1 an par défaut) peut être augmenté jusqu'à 10 ans pour réduire le temps de simulation.

Le germe d'initialisation des nombres pseudo-aléatoire est utilisé pour générer le peuplement et sélectionner les arbres à éclaircir. Par défaut il est modifié automatiquement à chaque simulation, mais peut être fixé en décochant la case 'Auto' pour répéter plusieurs fois la même simulation.

Format d'un fichier d'inventaire

Le fichier d'inventaire comprend un entête optionnel décrivant le peuplement suivi d'une liste d'arbres décrits par un identifiant unique et un diamètre (en cm) : 

# Stand data #
# (the settings given in the dialog box will override these values)
age = 100
subModel = Vosges
altitude_m = 500.0
hDomFert_m = 25.0
ageFert = 100
area_ha = 0.25
nbYearsByStep = 1
randomSeed = 0
randomSeedAuto = false

# Tree data #
# id	dbh
19177	12.249505996704102
19114	12.121687889099121
19017	11.922795295715332
19355	13.238288879394531
18991	11.87649154663086
19371	13.391877174377441
19009	11.91301441192627
18895	11.727401733398438
19349	13.14688491821289
19025	11.931548118591309

Un exemple de fichier d'inventaire peut être trouvé dans le répertoire capsis4/data/abial.

Pour créer un fichier d'inventaire à partir d'une simulation, cliquer avec le bouton de droite sur l'étape finale et choisir le menu Exporter puis le format Inventaire AbiAl.
