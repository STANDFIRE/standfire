Domaine de validité du modèle MELIES

Le modèle Melies a été calibré avec les données brutes des campagnes 2005 à 2008 de l'Inventaire Forestier National. La zone géographique de validité du modèle correspond aux 4 massif montagneux pour lesquels on trouve ce mélange en France : Vosges, Jura, Alpes et Massif Central.

Vous trouverez ci-dessous des informations détaillées par variable.

Pour de plus amples informations, voir Vallet et Pérot 2011, Journal of Vegetation Science, 22(5):932–942

Valeurs dendrometriques

Diamètre quadratique sapin et épicéa : de 7.5 à 60 cm de diamètre.

7.5 cm correspond au seuil de précomptage des données IFN.

Taux de mélange : de 0 à 1

Variables stationnelles

Température de mars :

de 0.425 à 4.8 °C

Précipitations d'avril :

de 77.3 à 147.9 mm

Pente :

de 0 à 35.8 degrés

Humus :

1 pour les sol de type Mull, 0 dans les autres cas.

Cependant, dans le cas d'une moyenne sur plusieurs station, la valeur peut être un réel entre 0 et 1. Dans ce cas, elle correspond à la proportion de placette de type Mull pour la population choisie.

Par exemple, les valeurs pré-enregistrées pour les massifs sont les proportions de placette IFN de type Mull dans la base “mélange sapin-épicéa”.

Quantité de terre fine

de 0.08 à 0.81 m3/m2

Taux d'affleurements rocheux

de 0 à 0.80 (80%)

Roche mère :

1 pour les roches mères de type calcaire, 0 dans les autres cas.

Cependant, dans le cas d'une moyenne sur plusieurs station, la valeur peut être un réel entre 0 et 1. Dans ce cas, elle correspond à la proportion de placette avec une roche mère de type calcaire pour la population choisie.

Par exemple, les valeurs pré-enregistrées pour les massifs sont les proportions de placette IFN de type calcaire dans la base “mélange sapin-épicéa”.

Température d'août :

de 13.5 à 17.4 °C

— Patrick Vallet 2011/09/09
