Objectifs

Le module AMAPsim est utilisé dans la connexion entre certains modules Capsis (PP3 - Pin Maritime…) et le simulateur AMAPsim. Il permet de charger des inventaires ou de construire des projets à partir des données retournées par AMAPsim.

Ce module ne comporte pas de méthode d'évolution. Pour plus de détails, consulter la documentation 'Connexion Capsis AMAPsim' sur le site Capsis.
