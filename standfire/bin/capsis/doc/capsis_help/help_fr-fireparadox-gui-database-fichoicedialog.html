MENU PRINCIPAL

La base de données EuroForestFuel permet de stocker différentes catégories de combustibles pouvant être ensuite placés sur des scènes de végétation virtuelles construites avec l'éditeur de scène.

Trois catégories de combustibles sont considérés :

  • Les plantes
  • Les nappes
  • Les échantillons de terrains

Tous ces combustibles peuvent être soit virtuels soit mesurés.

Les descriptions de combustibles sont fournis pas des équipes (essentiellement les partenaires du projet Fire Paradox) en provenance de sites ou de parcelles pouvant être géographiquement localisés.

Equipe etSite sont deux niveaux organisationnels de données qui sont aussi stockés dans la base de données.

Ce menu permet donc d'accéder, en respectant certains niveaux de droit, à la gestion des équipes, sites et différents types de combustibles.

GLOSSAIRE

Plante: objet de végétation.

Plante mesurée: objet de végétation correspondant à une plante réelle mesurée sur le terrain.

Plante virtuelle: objet de végétation ne correspondant pas à une plante réelle mesurée sur le terrain. Elle peux être différente par sa forme, par la distribution des biomasses dans sa forme, par les valeurs de ses différents paramètres.

Nappes: collection de plantes individuelles, regroupées de manière compacte, rendant difficile une description séparée. Une nappe sera décrite comme un objet unique ayant les même propriétés qu'une plante individuelle. Quercus coccifera est typiquement une nappe.

Echantillon: niveau élémentaire d'une description d'objet de végétation. Un ou plusieurs échantillons sont nécessaires à la construction d'un objet de végétation. Généralement la méthode utilisée pour échantillonner est la méthode “cubes” qui consiste à collecter des volumes élémentaires de 25 cm de cotés. Un échantillon type est donc un cube de 25 cm x 25 cm x 25 cm, mais ces dimensions peuvent variées. La valeur des paramètres d'un échantillon peuvent être mesurées de manière destructive ou être calculées.
