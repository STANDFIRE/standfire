2. Objectives

PP3 is a model of forest dynamics designed to the stand dynamics and silviculture interactions. It can be used to compare thinning strategies .

3. Description

PP3 module is based on a Distance-Independent tree Model.

4. Validity

Pp3 model was calibrated for Maritime pine stands located in Landes de Gascogne (except Dunes), France.
