Visibles steps

Principle

Choose the steps which will be visible in the project manager.

Method

All the steps are shown in this dialog box, including the ones not visible in the project manager.

Click on the step buttons to change their status. The red buttons are for the visible steps which will be visible in the project manager.

Note: some steps can NOT be set unvisible:

  • the root step of the project
  • the final steps of the scenarios
  • the steps with several daughters (nodes connected behind)
