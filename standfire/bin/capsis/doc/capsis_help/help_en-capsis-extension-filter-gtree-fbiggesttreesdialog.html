Selection of the n biggest trees

Select the n biggest trees according to their diameter at breast height (1.3m).

Type in the chosen number of trees, an integer greater than 1.

The 'Possible value' button will tell the maximum possible number that can be chosen.

Note: The filter sorts the trees on two criteria: decreasing diameter then decreasing identifier to answer in a consistant manner. Thus, if several trees have the same diameter, the n biggest returned will always be the same.
