LISTE DES COMBUSTIBLES

Tous les objets combustibles de type plantes ou nappes stockés dans la base de données apparaissent dans cette liste.

La liste présente pour chaque combustible, le code équipe, site, le nom de l'espèce, les dimensions du houppier, la référence ou ID interne, l'origine du combustible (virtuelle ou mesurée) et la mention si celui-ci est validé ou désactivé.

Les combustibles sont triés dans l'ordre : équipe - site - espèce - hauteur - diamètre.

Les sélecteurs en haut de l'écran permettent de restreindre la taille de cette liste.

Les actions possibles sont:

  • Ajouter: pour ajouter un nouveau combustible
  • Formes: pour gérer les différentes formes rattachées à ce combustible
  • Paramètres: pour modifier les paramètres relatifs à ce combustible
  • Valider/Dévalider: pour valider ou dévalider le combustible dans la base de données
  • Editer: pour éditer le combustible
  • Copier: pour copier le combustible dans un nouveau
  • Désactiver/Réactiver: pour désactiver ou réactiver le combustible
  • Fermer: pour fermer la fenêtre

La fonction Editer permet de voir les données de ce combustible. Les modifications sont possibles si l'utilisateur est propriétaire de la donnée et que le combustible n'est ni désactivé, ni validé. Dans le cas contraire, le combustible devra déjà être ré-activé ou dévalidé par son propriétaire ou l'administrateur pour être à nouveau modifiable.

La fonction Valider permet de vérifier l'intégrité des données relatives à un combustible dans la base de données et d'en permettre l'utilisation dans la création de scènes virtuelle et dans l'export vers des modèles comme FireTec. Pour être validée, une plante doit contenir au moins une forme 3D, et une nappe une forme 2D. Il n'est plus possible de modifier un combustible validé.

La fonction Dévalider permet de de lever la validation d'un combustible pour le mettre à jour.

La fonction Désactiver permet d'interdire l'utilisation d'un combustible sans pour autant détruire les données lui faisant déjà référence. La désactivation d'une plante entraine aussi la désactivation de toutes les formes qui lui sont rattachées.

La fonction Réactiver lève la désactivation du combustible mais pas des formes qui lui sont associées.
