Viewer for tree distance-independent models

Principle

View the distribution of trees per diameter classes and the height curve for the stand for the reference step and optionally the previous visible step.

Description

The histograms and height curves for the two steps are superposed. A table gives the main dendrometrical values for the two considered stands.

The viewer can e parametrized, especially :

  • aggregation according to a given class width or number of classes,
  • use of a minimum or maximum threshold for the representation,
  • choose girth or diameter mode,
  • restrict to a given pre-existing tree group,
  • compute all the values per hectare,
  • selection of the bars to view the properties of the underlying aggregated trees.
