FUEL EDITION

This interface aims at create, update or display fuel (plants or layers) in the database.

CREATION/UPDATE

Data entry is organized in 4 parts

  • Team
  • Site description
  • Simple individual
  • General comments

At the bottom of the screen, control boutons are :

Save in Database : to valid the new plant or modification

Cancel : to cancel and close the window

* PART 1 : TEAM

Data checking before validation are:

Team choice is compulsory

Sampling date has to be in the following format: DD/MM/YYYY

Fiels name operator 1 is compulsory

* PART 2 : SITE

Site choice is compulsory

Sites editor: to add or modify sites

* PART 3 : SIMPLE INDIVIDUAL

Species choice is compulsory

Latitude has to be numerical and between 0 and 360 degrees

Longitude has to be numerical and between 0 and 90 degrees

Altitude has to be numerical and unit is meters

Heights and diameters have to be numerical and unit is centimeters

Crown height and diameter max are compulsory

Total measured biomass has to be numerical and unit is grammes

Tree cover has to be numerical and between 0 and 100 %.

* PART 4 : GENERAL COMMENTS

DISPLAY
