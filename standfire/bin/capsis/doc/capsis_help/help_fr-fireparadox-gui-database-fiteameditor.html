EDITION DES EQUIPES

Cette interface permet de créer, modifier, désactiver ou réactiver des équipes dans la base de données.

CREATION D'UNE EQUIPE

Entrez ici le nom de l'équipe et le mot de passe de la nouvelle équipe. Ces 2 champs sont obligatoires.

Le mot de passe permet aux membres de l'équipe de se connecter à la base pour créer, modifier ou désactiver des données appartenant à sa propre équipe.

Par contre, toutes les données seront en libre accès, en visualisation, à tous les partenaires du projet FireParadox.

  • Enregistrer dans la base : Sauvegarde la nouvelle équipe et ferme la fenêtre
  • Annuler : Abandonne la saisie et ferme la fenêtre
MODIFICATION D'UNE EQUIPE

Entrez ici le nouveau nom si vous souhaitez le modifier. Ce champs est obligatoire.

Sélectionnez une personne dans la liste et cliquez sur Gérer les personnes pour modifier son nom ou la désactiver.

Sélectionnez New person dans la liste et cliquez sur Gérer les personnes pour créer une nouvelle personne dans cette équipe.

  • Gérer les personnes : pour ajouter ou modifier les personnes de cette équipe
  • Mot de passe : pour changer le mot de passe de cette équipe
  • Enregistrer dans la base : Sauvegarde les changements et ferme la fenêtre
  • Annuler : Abandonne les changements et ferme la fenêtre
DESACTIVATION D'UNE EQUIPE

Le nom de l'équipe apparait à l'écran pour vérification.

Après validation, l'équipe ne sera plus autorisée à gérer des données dans la base.

L'équipe ne sera pas détruite physiquement de la base de données mais seulement désactivée logiquement.

Toutes ses données (plantes, nappes, échantillons) seront conservées et pourront être utilisées dans le Fuel manager.

  • Désactiver dans la base : Valide la désactivation.
  • Annuler : Abandonne et ferme la fenêtre
REACTIVATION D'UNE EQUIPE

Le nom de l'équipe apparait à l'écran pour vérification.

Après validation, l'équipe sera de nouveau autorisée à gérer des données dans la base.

  • Réactiver dans la base : Valide la réactivation.
  • Annuler : Abandonne et ferme la fenêtre
