LISTE DE FORMES POUR UN COMBUSTIBLE

Tous les formes rattachées à un même combustible et stockées dans la base de données apparaissent dans cette liste.

Elle contient le type de forme (plante, nappe ou échantillon), l'information XZ, XZ_XY ou XYZ indiquant si la forme est en 2D, 2 fois 2D ou 3D, la méthode d'échantillonnage utilisée (cube ou cage) les dimensions du houppier et la mention si celle-ci est désactivée.

En haut de l'écran à gauche, un cadre rappelle les informations générales concernant le combustible.

Les actions possibles sont:

  • Ajouter: pour ajouter une nouvelle forme à cette plante
  • Editer: pour éditer une forme
  • Désactiver/Réactiver: pour désactiver ou réactiver une forme
  • Fermer: pour fermer la fenêtre

La fonction Ajouter n'est possible que si l'utilisateur est propriétaire de la plante et si celle ci n'est ni désactivée, ni validée. De plus il est impossible de saisir 2 formes 3D actives pour une même plante ou 2 formes 2D actives pour une même nappe.

La fonction Editer permet de voir la forme. Les modifications seront possibles si l'utilisateur est propriétaire de la plante et si celle ci n'est ni désactivée, ni validée.

La fonction Désactiver permet de suspendre l'utilisation d'une forme sans pour autant la détruire physiquement dans la base de données. Cette fonction n'est possible que si l'utilisateur est propriétaire de la plante et si celle ci n'est ni désactivée, ni validée.

Plantes mesurées

Si l'échantillonnage a été réalisé avec la méthode cube, alors la liste des formes est composée d'un échantillon SAMPLE UNIQUE et d'une forme 3D. Ensuite il est possible d'avoir une forme 2D ou une forme 2 * 2D ayant servi d'intermédiaire pour la construction de la forme 3D.

Si l'échantillonnage a été réalisé avec la méthode cage, alors les liste des formes est composé d'un échantillon SAMPLE UNIQUE et d'une forme 3D. Il n'est pas possible dans ce cas d'avoir des formes 2D.

Plantes virtuelles

Pour les plantes virtuelles, la présence d'un échantillon est optionnel. Si un échantillon virtuel a été crée, alors il est obligatoirement de méthode cube.

Nappes mesurées

Il n'est pas possible d'utiliser la méthode cage pour les nappes.

La liste des formes est alors composée d'un ou deux échantillons SAMPLE CORE et/ou SAMPLE EDGE et d'une forme 2D composée d'une partie coeur et/ou d'une partie bord.

Nappes virtuelles

Pour les nappes virtuelles, la présence d'un échantillon est optionnel.
