Selection of trees by ground cells

Select some trees by selecting the ground cells containing them.

Customizing the display

Change the number of displayed labels by changing the value and validate. Change the magnify factor for the trees diameter in the same manner. Replace the trees identifiers by their diameters at 1.3 m with the diameter check box. Change the level at which the cell selection will occur in case of several levels of nested cells.

Selection / Deselection of cells

To (de)select a cell, double-click on it (left button) To (de)select a group of cells: define a selection rectangle with the right button

To deselect all, use the red button

Information about one or more trees

Select the tree(s) while holding down the Control key.

Zoom

Zoom in: define a zoom rectangle with the left button Zoom out: right-click on the image

Move on the image

Draw a translation vector on the image while holding down the Control key.
