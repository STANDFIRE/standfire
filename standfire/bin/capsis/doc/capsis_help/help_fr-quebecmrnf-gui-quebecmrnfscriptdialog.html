Exécution du script

Ce menu vous permet d'exécuter un script, c'est-à-dire d'exécuter plusieurs simulations lorsque votre fichier d'inventaire comprend plusieurs strates. Pour exécuter un script, 4 étapes sont préalables. Vous devez dans l'ordre :

  1. Lire votre fichier d'inventaire.
  2. Lire les paramètres d'évolution.
  3. Choisir un niveau d'exportation (arbres individuels, placettes ou strate).
  4. Choisir un fichier d'extrants dans lequel les prévisions seront exportées.

Ce menu vous guide à travers ces étapes avant d'exécuter le script.
