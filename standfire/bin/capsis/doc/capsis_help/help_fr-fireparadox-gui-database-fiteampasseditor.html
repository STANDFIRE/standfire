MODIFICATION DU MOT DE PASSE D'UNE EQUIPE

Cette interface permet à une équipe de changer son propre mot de passe.

L'administrateur de la base a aussi le droit de changer les mots de passe de toutes les équipes.

Le nom de l'équipe apparait pour vérification.

L'ancien mot de passe doit obligatoirement être fourni si l'utilisateur n'est pas l'administrateur de la base.

Le nouveau mot de passe est obligatoire.

  • Enregistrer dans la base : Sauvegarde le nouveau mot de passe et ferme la fenêtre
  • Annuler : Abandonne la saisie et ferme la fenêtre
