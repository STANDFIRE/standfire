EDITION DES EQUIPES

Toutes les équipes répertoriées dans la base de données apparaissent dans cette liste.

Elle contient le nom des équipes et la mention désactivée si l'équipe n'est plus active dans le projet.

Les actions possibles sont:

  • Modifier: pour modifier une équipe existante
  • Ajouter: pour ajouter une nouvelle équipe
  • Désactiver/Réactiver: pour désactiver ou réactiver une équipe
  • Fermer: pour fermer la fenêtre

La fonction désactiver permet de supprimer tous les droits d'accès d'une équipe sans pour autant détruire les données qui lui appartiennent.
