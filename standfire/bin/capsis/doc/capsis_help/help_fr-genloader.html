Module GenLoader

Auteur

François de Coligny (INRA), UMR AMAP, botAnique et bioinforMatique de l'Architecture des Plantes, TA 40 / PS2 - Boulevard de la Lironde, 34398 Montpellier Cedex 5

Objectifs

GenLoader est un module technique sans procédure d'évolution. Il est destiné au chargement d'inventaires spatialement explicites dans l'étape initiale d'un projet pour évaluation ou visualisation avec des outils Capsis.

Il a été initialement développé pour tester les visualiseurs 3D de Capsis. Il peut cependant être utilisé à d'autres fins.

Description

GenLoader est un dispositif de chargement de fichiers inventaires forestiers spatialisés simples.

Entrées

GenLoader utilise des fichiers inventaires décrivant les caractéristiques de chaque arbre : id, x, y, z, dbh (cm.), height (m.), crownBaseHeight (m.), crownRadius (m.), stateCode, shapeCode, transparency

Notes :

id est un entier

stateCode est un entier (correspondant à une couleur) 

stateCode :
0 - "cible", rouge
1 - jaune
2 - gris
i - vert

shapeCode est 'e' (ellipsoid) ou bien 'c' (cone)

les autres valeurs sont décimales

transparence: 0.0 (opaque) à 1.0 (transparent)

Exemple d'inventaire avec 4 arbres :

21 0 40 0 10 10 5 2.5 3 e 0

22 0 50 0 10 10 5 2.5 3 c 0

23 50 50 0 10 10 5 2.5 3 e 0

24 50 40 0 10 10 5 2.5 3 c 0

Sorties

Des Visus peuvent être utilisés pour examiner les caractéristiques du peuplement chargé (Visu Simple, Explorateur, Visu 3D…).
