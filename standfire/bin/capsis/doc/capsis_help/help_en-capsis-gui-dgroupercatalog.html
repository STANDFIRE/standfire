Group Catalog

Principle

Manage the groups (trees, cellsâ€¦) from this catalog.

Possible actions

Create a new group, modify an existing group or remove a group with the proposed actions. The actions apply on the groups of the type currently selected in the upper combo box. This combo box contains the available group types compatible with the model associated with the current step.

Complex groups

Complex groups can be built by combining the existing groups with the logical operation OR (union of the elements of the two selected groups) or AND (intersection of the elements of the two selected groups). The new complex groups are added to the available group list and can be used in the same conditions than the simple groups.

Note: building comlpex groups can slow the application by raising the complexity of the combined operations when using them in the Capsis tools and extensions.
