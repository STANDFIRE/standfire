Nouvelle Session

Paramétrage

Choisissez un nom pour la session, puis créez un projet dans la foulée ou par le menu “Projet > Nouveau”.

Remarque : si des projets sont créés directement par “Projet > Nouveau”, ils sont implicitement contenus dans une session par défaut nommée “untitled”. La gestion par session est optionnelle, elle permet notamment de sauvegarder tous les projets en cours en une seule opération parFichier > Enregistrer Session.
