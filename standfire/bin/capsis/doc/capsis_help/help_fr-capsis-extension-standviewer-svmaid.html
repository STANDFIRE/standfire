Visualiseur Modèles Arbre Indépendant des Distances

Principe

Visualisez la distribution des arbres par classes de diamètre ainsi que la courbe de hauteur du peuplement pour l'étape de référence et la précédente étape visible.

Description

Les histogrammes et les courbes de hauteur pour les deux étapes sont superposés. Une table donne les valeurs des principales grandeurs dendrométriques pour les deux peuplements considérés.

Le visualiseur est paramétrable, on peut noter :

  • aggrégation selon une largeur de classe ou un nombre de classes donné,
  • imposition d'un seuil minimum et / ou maximum pour la représentation,
  • choix du mode circonférence / diamètre,
  • restriction à un groupe d'arbres préétablis,
  • valeurs par hectare,
  • sélection des barres pour consulter les propriétés des arbres types aggrégés sous-jacents.
