EVENTS EDITION

This interface aims at create, modify, delete events attached to a site in the database.

EVENT CREATION OR MODIFICATION

Before validation data checking are :

Type is compulsary

Starting date is compulsary and format have to be DD/MM/YYYY

Ending date is compulsary and format have to be DD/MM/YYYY

Ending date have to be greather or equal to starting date.

  • Save : Save the new event or modifications
  • Cancel : Cancel modifications and close the window
EVENT SUPRESSION

Event data appears on this screen for checking.

After validation, event will be deleted in the list.

  • Delete : Valid the supression.
  • Cancel : Cancel supression and close the window
