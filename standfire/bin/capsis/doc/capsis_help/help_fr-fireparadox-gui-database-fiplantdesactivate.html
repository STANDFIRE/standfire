DESACTIVATION OU REACTIVATION D'UN COMBUSTIBLE

Cette interface permet de désactiver ou réactiver un combustible de type plante ou nappe dans la base de données.

Ces actions ne sont possibles que sur les combustibles dont l'utilisateur est le propriétaire.

DESACTIVATION

Les données relatives au combustible apparaissent à l'écran pour vérification.

Après validation, le combustible ainsi que toutes les formes rattachées (échantillons, 2D, 3D) ne seront plus accessibles dans la base pour la création de scène virtuelle ou pour l'exportation dans des modèles comme Firetec.

Le combustible ne sera pas détruit physiquement de la base de données mais seulement désactivé logiquement.

  • Enregistrer dans la base : Valide la désactivation.
  • Annuler : Abandonne et ferme la fenêtre
REACTIVATION

Les données relatives au combustible apparaissent à l'écran pour vérification.

Après validation, le combustible sera de nouveau accessible dans la base.

Seule la plante ou la nappe sera réactivée, mais pas les formes !

  • Enregistrer dans la base : Valide la réactivation.
  • Annuler : Abandonne et ferme la fenêtre
