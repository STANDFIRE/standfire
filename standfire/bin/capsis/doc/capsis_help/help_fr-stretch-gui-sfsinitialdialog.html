Initialisation d'un projet Stretch

Choisissez une bibliothèque d'espèces ou créez des espèces, choisissez les paramètres généraux, puis passez à a plantation des arbres avec le bouton Suivant. Une fois les arbres plantés, validez la création de l'étape initiale du projet par Ok.

Alternativement, utilisez l'Onglet Import Etape Stretch pour recharger une étape préalablement exportée avec l'Export Etape Stretch (sur une étape de scénario, Menu contextuel > Exporter > Export d'une étape d'un projet Stretch). Au cours de cet import, il est possible de changer l'option Stretch (Lieberman avec ou sans couronne, MMR, SLIM) ainsi que les valeurs des paramètres des espèces. Les arbres sont replantés aux mêmes positions.

Onglet Espèces

Création d'une espèce : Le bouton Créer ouvre un éditeur d'espèces. Renseigner les champs vides et valider. La nouvelle espèce est ajoutée à la bibliothèque. Pour Enregistrer la biliothèque sur disque, utilizer Enregistrer Sous.

Chargement d'une bibliothèque d'espèces : Le bouton Charger ouvre un dialogue pour choisir un fichier bibliothèque d'espèce préalablement sauvegardé.

Choisissez Editer pour modifier les paramètres de l'espèce sélectionnée et Supprimer pour l'enlever de la bibliothèque. Enregistrez vos modifications sur disque pour mettre à jour la bibliothèque sur disque.

Le nom du Fichier espèces en cours d'utilisation apparait au bas de l'onglet.

Onglet Paramètres généraux

Choisissez l'option du modèle Stretch parmi Lieberman avec ou sans couronne, MMR (Mir Musc Radbal) et SLIM. Renseignez les paramètres de taille du terrain.

Il vous est possible d'enregistrer les paramètres actuels sur disque ou de recharger un fichier de paramètres précédement enregistré.

Onglet Import Etape Stretch

Activez le processus d'import, puis choisissez le fichier à importer et chargez le. Ce fichier a été produit au cours d'une simulation préalable à l'aide de l'Export Etape Stretch.

A ce stade, vous pouvez contrôler le contenu du fichier (valeurs des paramètres, bibliothèque d'espèces, positions des arbres…). Il vous est possible de modifier certains paramètres pour la nouvelle simulation sur les Onglets Espèces et Paramètres Généraux (Option Stretch, valeurs de paramètres pour certaines espèces). Certains de ces paramètres sont bloqués (taille du terrain, taille de cellule…).

Ensuite, Validez l'import ce qui a pour effet de planter les arbres, puis terminez par Ok.

Attention: Si la bibliothèque d'espèce n'est pas trouvée (déplacée, renommée, import effectué sur une autre machine…), vous êtes prévenus. Vous avez la possibilité de créer les espèces requises (la liste vous est donnée) ou de charger vous même une bibliothèque d'espèces qui les contient.

Onglet Plantation aléatoire

Choisissez parmi les espèces de la bibliothèque sélectionnée sur l'onglet Espèces celles que vous souhaitez planter et passez les dans la liste de droite avec les boutons fléchés. Pour chacune de ces espèces, éditez la colonne Nombre d'arbres pour spécifier combien d'individus vous souhaitez planter. Enfin utilisez Planter pour procéder à la plantation. La localisation des arbres est choisie aléatoirement.

Validation de l'étape initiale

Une fois les arbres plantés à votre convenance, validez le dialogue initial par Ok. Le projet s'ouvre dans le Gestionnaire de projets.
