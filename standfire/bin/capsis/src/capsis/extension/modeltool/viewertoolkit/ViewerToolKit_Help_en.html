Principle

Select the tree of interest and view them with the compatible viewers.

It is possible to restrict the list of trees to a given group, the list is updated each time the "Group" component is changed.

Select one or more trees in the table and they will appear in the viewer chosen in the related combo box. To select several trees, use the ctrl and shift keys while selecting. Ctrl-a selects all the trees in the table.

Some viewers can only show one tree, in case of multiple selection they will show the first tree.
