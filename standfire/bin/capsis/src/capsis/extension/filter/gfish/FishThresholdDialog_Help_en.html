Selection of fish with threshold

 Capsis Help

Principle

Select fish by specifying low / high thresholds on their numeric properties.

  • Choose the property (ex: fork length)

  • Enter a low threshold for the property

  • Enter a high threshold

Both thresholds are optionnal : if not specifyed, the low threshold is considered equal to zero and the high threshold is set to the max possible value to have no effect.

It is thus possible to specify "more than value a", "less than value b" or "between value a and value b".
