Numeric thresholds filter

 Capsis Help

Principle

Activate some numeric properties and set low and high thresholds for them. The selected elements will match these constraints.

Proposed properties

The list contains all the numeric properties common to all the elements in the collection to be filtered.

The collection may contains elements of different types, for example several types of tree descriptions with a common part and some particularities. Only the common properties are considered here.

Only the activated properties are taken into account. To activate a property, check the first check box of the line: the name of the property becomes enabled. The text fields are used to set the thresholds for this variable.

The thresholds must all be numbers, the low value is included, the high value is excluded.

If several properties are activated, the filter checks them by processing a logical AND: the kept elements will match all the thresholds.
