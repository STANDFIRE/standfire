Sélection individuelle

 Aide Capsis

Principe

Sélectionnez des éléments grâce à leurs identifiants : désignez des éléments dans la liste de gauche et faites les passer dans la liste de droite. Ils sont alors sélectionnés.

Désignation des éléments

Cliquez sur un éléments d'une liste pour le désigner.

Ensuite, éventuellement :

  • cliquez un autre élément en maintenant Shift enfoncé pour désigner un groupe d'éléments contigus

  • cliquez d'autres individus de la liste en maintenant Control enfoncé pour désigner plusieurs éléments non contigus

Passage d'une liste à l'autre

Ajoute les éléments désignés à gauche dans la liste de droite
Enlève les éléments désignés dans la liste de gauche
Ajoute tous les éléments de la liste de gauche dans la liste de droite
Enlève tous les éléments de la liste de droite
Trie les deux listes dans l'ordre ascendant
