Filtre sur seuils numériques

 Aide Capsis

Principe

Activez les propriétés numériques de votre choix et fixez des seuils bas et haut pour ces propriétés. Les éléments sélectionnés respecteront ces seuils.

Propriétés proposées

La liste contient les propriétés numériques communes à tous les éléments de la collection à filtrer.

Il est possible que la collection contienne des éléments de plusieurs types différents, par exemple plusieurs types d'arbres présentant une base de description commune et des particularités pour chacune. Seules les propriétés numériques communes sont considérées par ce filtre.

Seules les propriétés activées sont prises en compte. Pour activer une propriété, cocher la première case de la ligne : le nom de la propriété devient actif. Les zones de texte sont utilisées pour spécifier les valeurs souhaitées pour les seuils bas et haut pour cette variable.

Les seuils doivent tous être des nombres, le seuil bas est inclu, le seuil haut est exclu.

Si plusieurs propriétés sont activées, le filtre les vérifie toutes en effectuant un ET logique : les éléments résultants vérifieront tous les seuils.
