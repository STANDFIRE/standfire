Aide Capsis

Principe

Ce visualiseur est compatible avec les modèles qui gèrent une liste d'arbres (individus ou cohortes). Il montre une table contenant une ligne par structure de données arbre présent dans la scène. Pour chaque arbre, les colonnes suivantes sont proposées : identifiant, âge, diamètre, hauteur, nombre, marqué, x, y, z, espèce. Chacune des colonnes est renseignée si possible : si les arbres du modèle ne contiennent pas l'une des propriétés correspondantes, la colonne est laissée vide.

Mode d'emploi

L'effectif total est cumulé dans le champ « N ». Le nombre d'objets arbres est présenté dans le champ « lignes ». Les deux valeurs correspondantes sont égales dans le cas d'individus, elles peuvent être différentes dans le cas de cohortes où un objet arbre peut représenter plusieurs individus, voir la colonne « Nombre ».

On peut trier la liste d'arbres dans l'ordre ascendant sur la propriété correspondant à une colonne en cliquant sur l'entête de la colonne. Un ctrl-clic trie dans l'ordre descendant.

Il est possible de restreindre la liste aux arbres d'un groupe en paramétrant le composant « groupe ». La liste est mise à jour en direct à chaque modification.

Il est également possible de sélectionner des arbres dans la liste est de les observer plus en détail en choisissant un mode de sélection dans la liste proposée et en actionnant le bouton « montrer la sélection ». Quand un arbre est sélectionné dans la table, une description sommaire sous forme de texte est proposée dans le champ description au bas du visu.
