The Samsaralight library (class prefix SL)


Author: Benoit Courbaud, Cemagref

Benoit.Courbaud@grenoble.cemagref.fr


Implementation: Nicolas Donès, François de Coligny - october 2008

package: capsis.lib.samsaralight


To use this library, a capsis module must follow some rules:

An individual based, spatialy explicit model with square cells and

- the stand must be a GTCStand

- the trees must be GTree instances implementing Spatialized, SLLightableTree and Speciable

- the plot must be a RectangularPlot implementing SLLightablePlot

- the cells must be SquareCell instances implementing SLLightableCell


How to use it


1. In the constructor of the Model class,

slModel = new SLModel ();


2. in model.initialize ()

slModel.computeRelativeCellNeighbourhoods (initStand, treeMaxHeight, bottomAzimut, cellWidth, maxCrownRadius);

// Initial radiative balance

slModel.processLighting (initStand);
