Description   

This library offers the possibility to create and manage an additional data structure based on the characteristics of the trees in the studied population in order to calculate their biomechanical behaviour under various loadings, in particular drag forces due to wind. Moreover, it contains the set of graphic tools allowing results of structures computation to visualize by different ways.

Furthermore, Biomechanics is the tools provider for the FOREOLE Model (intervener) and the Analyser FOREOLE (model tool)...
