Loxodonta viewer

Capsis Help

Principle

View the spatialized stand under the reference step as a map. This viewer is based on the Simple viewer and has the same main features. It is available for most of the forestry models using the genetics library.

Features

Each tree is represented by a point which is drawn at its real position in the stand. The map can show individuals as well as cohorts. In this case, the tree is drawn in its cell and it represents a number of trees equal to the value of its "number" property.
