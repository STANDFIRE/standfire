Exportation de Bois mort

 Aide Capsis

Principe

Exportation des valeurs de bois mort d'origine anthropique

Pour les équations, voir  :
Brin A., Meredieu C., Piou D., Brustel H., Jactel H. 2008. Changes in quantitative patterns of dead wood in maritime pine plantations. Forest Ecology and management, 256, 913-921.

Valeurs fournies en m3/ha

Description des colonnes

  • Date : Age du peuplement
  • Thi.Exp. Log : Volume de billons exportés (dimensions : Longueur : 2.10 m - Circonférence fin bout : 20 cm) 

  • Thi.DW Log : Volume total de billons laissé dans le peuplement lors de l'éclaircie

  • Thi.DW Log < : Volume de billons laissé dans le peuplement lors de l'éclaircie (catégorie "petit bois" - ventilation choisie dans la boîte de dialogue d'initialisation de la simulation)

  • Thi.DW Log > : Volume de billons laissé dans le peuplement lors de l'éclaircie (catégorie "gros bois" - ventilation choisie dans la boîte de dialogue d'initialisation de la simulation)

  • Thi.DW Stump : Volume de souche laissé dans le peuplement lors de l'éclaircie

  • Thi.DW Stump < : Volume de souche laissé dans le peuplement lors de l'éclaircie (catégorie "petit bois" - ventilation choisie dans la boîte de dialogue d'initialisation de la simulation)

  • Thi.DW Stump > : Volume de souche laissé dans le peuplement lors de l'éclaircie (catégorie "gros bois" - ventilation choisie dans la boîte de dialogue d'initialisation de la simulation)

  • Thi.DW Branch : Volume de branches laissé dans le peuplement lors de l'éclaircie

  • DW total : Volume total de bois mort laissé dans le peuplement par les éclaircies et prise en compte de la dégradation

  • DW Log : Volume total de billons laissé dans le peuplement par les éclaircies et prise en compte de la dégradation

  • DW Stump : Volume de souche laissé dans le peuplement par les éclaircies et prise en compte de la dégradation

  • DW Branch : Volume de branches laissé dans le peuplement par les éclaircies et prise en compte de la dégradation
