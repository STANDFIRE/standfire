Description   

Type in Capsis : Model Tool

Also view: FOREOLE Model (intervener)

The analyser FOREOLE allows wind sensitivity of a forest stand to be evaluated with time. For this, it uses the functions of the FOREOLE Model and of the Biomechanics library  at each step (from the root step to the selected current). At the end, it proposes a curve of the temporal evolution of stand resistance...
