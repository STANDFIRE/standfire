Description   

Type dans Capsis : Interveneur

Voir aussi : Analyseur FOREOLE (outil de modèle)

Le modèle FOREOLE permet de calculer les effets d'une perturbation naturelle liée au vent sur un peuplement forestier géré par un modèle de croissance ou de dynamique de type arbre compatible (en bref, l'objet "peuplement" doit être une instance de l'interface TreeCollection et l'objet "arbre" une instance de l'interface MecaTreeInfo :-). Une fois les calculs terminés, le choix est donné d' Appliquer la perturbation ou d' Annuler ... Le modèle est individu centré et permet ainsi d'évaluer la quantité de dégâts partiels, dus à l'application d'un vent moyen, au sein d'une population d'arbres. Les calculs et la visualisation des résultats reposent sur les fonctionnalités de la bibliothèque Biomechanics ...
