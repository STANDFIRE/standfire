Dialog with FOREOLE

Help on the dialog box Tags

  • Structures construction

  • Structures computation

  • Results access

NB: When the FOREOLE model is called from the current step, the structures construction is automatically launched with default parameters. Thus, the dialog box appears on the tag Structure computation. It is possible to reconstruct the structures with new mechanical parameters before to define wind profile and to launch the computation. Moreover, after a calculation, structures have to be reconstructed before to launch a new one...

Tag Structures construction   

Before the construction of biomechanical structures, adjust parameters to use:

    - wood volumic density

    - its modulus of elasticity

    - ratio between crown mass and stem mass

    - drag coefficient used to evaluate wind forces

These parameters are species dependent... Click on Parameters button.

Thus, the Creation button allows the construction of biomechanical structures to be launched in order to analyse the stand...

Tag Structures computation   

Before the calculation, it is possible to adjust wind loading parameters. Click on Parameters button. The wind speed can be specified (choosing, at stand edge, the corresponding reference height) and the location of the population in a theoretical larger stand can be choose: at stand edge or inside the stand. Parameters associated to wind profile have to be modified with caution...

Thus, the Launch button executes the calculation of the mechanical state of each tree submitted to the wind profile above defined. Moreover, this calculation is automatically followed by a damage analysis based on failure criteria at the tree level...

The Export button allows trees characteristics to be written in the Capsis LOG. It could take many time...

For more details on wind profile, method of structures computation and failure criteria (...), consult FOREOLE References.

Tag Results access   

The Inspect button opens an inspector on the object of highest level in the library Biomechanics (instance of MecaProcess), from which access to the collection of  trees (instances of MecaTree) is given, then, for example, to their elements (instances of MecaGU, corresponding to Growth Units)...

The % Damage button gives amount of partial wind damage relative to number of trees and to volume of wood in the population...

The View (2D) button opens the Cartographic Viewer 2D of Biomechanics on the stand...

The Points set button opens a graph where the tree level data represented in Abscissa and Ordinate have to selected in the list boxes below. In this graph, each point is associated to one tree and a colour legend indicates the tree state*. Moreover, double-click on a point opens an inspector on corresponding tree (functions relative to mouse are of the same type than those described for the Cartographic Viewer 2D and the Tree Viewer)...

The Histogram button allows the distribution of a tree level characteristics to be represented according to the selection made in the list box below. A colour legend also indicates the tree state* according to the chosen distribution...

 

*Tree state:

    - No Damage

    - Windthrow

    - Stem Breakage
