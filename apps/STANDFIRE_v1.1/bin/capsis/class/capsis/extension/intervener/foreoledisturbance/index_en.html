Description   

Type in Capsis : Intervener

Also view: Analyser FOREOLE (model tool)

The FOREOLE model allows wind effects (considered as a natural disturbance) to be calculated on forest stands managed by a tree-based model of forest growth or dynamics, which is compatible (i.e. the object "stand" has to be an instance of the TreeCollection interface and the object "tree" an instance of the MecaTreeInfo interface  :-). Thus, choose Apply or Cancel the disturbance... The model is tree-based, allowing the partial wind damage due to a mean wind loading to be evaluated in the population of trees. The computation and the visualisation of results are based on the functions of the Biomechanics library...
