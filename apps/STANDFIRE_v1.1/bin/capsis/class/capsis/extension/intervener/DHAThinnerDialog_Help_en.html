Thinning with thresholds

 Capsis Help

Principle

Select the trees to be cut by specifying low and/or high thresholds on their age, diameter or height.

  • Choose Age, Diameter or Height
  • Enter a low threshold for the considered property
  • Enter a high threshold

Both thresholds are optionnal: if not specified, the low threshold is considered equal to zero and the high threshold is set to the maximum possible value in order to have no effect.

It is thus possible to select "more than value a", "less than value b" or "between value a and value b".
