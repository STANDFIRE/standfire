Filtre vrai/faux

 Aide Capsis

Principe

Activez et positionnez à vrai ou faux les propriétés de votre choix pour sélectionner les éléments ayant les mêmes valeurs pour ces propriétés.

Propriétés proposées

La liste contient les propriétés booléennes (vrai/faux) communes à tous les éléments de la collection à filtrer.

Il est possible que la collection contienne des éléments de plusieurs types différents, par exemple plusieurs types d'arbres présentant une base de description commune et des particularités pour chacune. Seules les propriétés booléennes communes sont considérées par ce filtre.

Seules les propriétés activées sont prises en compte. Pour activer une propriété, cocher la première case de la ligne : le nom de la propriété devient actif. La deuxième case est utilisée pour spécifier la valeur souhaitée pour la variable : vrai (cochée) ou faux (décochée).

Si plusieurs propriétés sont activées, le filtre les vérifie toutes en effectuant un ET logique : les éléments résultants vérifieront toutes les propriétés.
