SITE EVENTS MANAGEMENT

This screen aims at manage all events attached to a site.

Events have a type, a starting date and a ending date.

Actions on buttons are:

  • Add: to add a new event
  • Update: to modify an event
  • Delete: to delete an event
  • Validation: to save the new event list
  • Cancel: to close the window without saving modifications
