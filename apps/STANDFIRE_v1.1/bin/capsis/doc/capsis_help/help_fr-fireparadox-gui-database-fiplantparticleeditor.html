SAISIE DES PARAMETRES DE PARTICULE POUR UN COMBUSTIBLE

Cette interface permet de créer, modifier, supprimer des valeurs de paramètres de particules pour un combustible.

En haut à gauche, un encadré reprend les informations principales relatives à la plante ou nappe.

Puis un tableau permet de saisir les valeurs des paramètres par famille de particules vivantes et mortes.

Un sélecteur permet de réduire cette liste à un seul paramètre si nécessaire.

La liste des paramètres disponibles est :

  • MVR en kg/m3
  • SVR en m2/m3
  • AC (Ashes content) en g/100g
  • MC (Moisture content) en %
  • HCV en KJ/Kg
  • Size (Taille) en mm

Les familles de particules correspondent aux biomasses qui ont été saisies dans les cubes représentant la forme associée à ce combustible.

La valeur 0.0 correspond à une valeur mesurée et égale à zéro.

La valeur -9.0 correspond à une valeur manquante non mesurée.

La valeur NaN correspond à une valeur existante mais non mesurée.

Les actions possibles sur les boutons :

  • Enregistrer dans la base : valide les modifications dans la base de données
  • Annuler : annule la saisie et ferme la fenêtre
