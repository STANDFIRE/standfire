EDITION DES MUNICIPALITES

Cette interface permet de créer, modifier, désactiver ou réactiver des municipalités dans la base de données.

CREATION D'UNE MUNICIPALITE

Saisir le nom de la nouvelle municipalité ainsi que le code pays. Ces deux champs sont obligatoires.

  • Enregistrer dans la base : valide la nouvelle municipalité
  • Annuler : annule la saisie et fermer la fenêtre
MODIFICATION D'UNE MUNICIPALITE

Saisir le nouveau nom de la municipalité ou changer le code pays. Ces deux champs sont obligatoires.

  • Enregistrer dans la base : valide les modifications
  • Annuler : annule les modifications et fermer la fenêtre
DESACTIVATION D'UNE MUNICIPALITE

Le nom de la municipalité apparait à l'écran pour vérification.

Après validation, la municipalité ne sera plus accessible dans la base.

La municipalité ne sera pas détruite physiquement de la base de données mais seulement désactivée logiquement.

Toutes les sites lui faisant référence seront conservées et pourront être utilisées dans le Fuel manager.

  • Désactiver dans la base : Valide la désactivation.
  • Annuler : Abandonne et ferme la fenêtre
REACTIVATION D'UNE MUNICIPALITE

Le nom de la municipalité apparait à l'écran pour vérification.

Après validation, la municipalité sera de nouveau accessible dans la base.

  • Réactiver dans la base : Valide la réactivation.
  • Annuler : Abandonne et ferme la fenêtre
