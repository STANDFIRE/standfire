Eclaircisseur Fagacée-Sylvestris

Cet éclaircisseur est basé sur RDIAutoThinner et permet d'appliquer ce dernier indépendamment sur le peuplement chêne et le peuplement pin.

Lors de l'application de l'eclaircisseur, le peuplement est dupliqué. D'un coté les pins sont supprimés et de l'autres les chênes. On obtient ainsi deux peuplements mono-essence sur lesquels un RDIAutoThinner différent est appliqué. Les peuplement résultants sont ensuite fusionnés.

Indiquez les valeurs de RDI souhaitées pour chaque essence ainsi que les coefficients d'éclaircie correspondants. Les tableaux indiquent les caractéristiques du peuplement lorsque l'intervention aura été réalisée avec ces paramètres.

Attention : les valeurs indiquées sont à l'ha, contrairement à la boite de dialogue initiale de fasy
