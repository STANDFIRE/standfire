Exportation

Principe

Choisissez un format et un nom de fichier pour exporter des données à partir de l'étape de référence.

Démarche

Les formats sont disposés dans une liste combinée. La sélection d'un format fait apparaître des informations le concernant.

Le fichier cible contiendra les données exportées. Choisissez une localisation sur disque et un nom pour ce fichier.

Une fois vos choix effectués, l'appui sur OK déclenche l'exportation et la création du fichier.

Remarque : Les formats peuvent générer suivant les cas :

  • une description du peuplement sous l'étape de référence,
  • une description concernant plusieurs étapes du scénario défini par l'étape racine et l'étape de référence.
