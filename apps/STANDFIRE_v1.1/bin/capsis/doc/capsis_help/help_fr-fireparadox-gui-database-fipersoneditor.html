EDITION DES PERSONNES D'UNE EQUIPE

Cette interface permet de créer, modifier, désactiver ou réactiver les personnes au sein d'une équipe.

CREATION D'UNE PERSONNE

Entrez ici le nom de la nouvelle personne. Ce champ est obligatoire.

  • Enregistrer dans la base : Sauvegarde la nouvelle personne et ferme la fenêtre
  • Annuler : Abandonne la saisie et ferme la fenêtre
MODIFICATION D'UNE PERSONNE

Entrez ici le nouveau nom si vous souhaitez le modifier. Ce champs est obligatoire.

  • Désactiver/Réactiver dans la base : Pour désactiver ou réactiver cette personne
  • Enregistrer dans la base : Sauvegarde les changements et ferme la fenêtre
  • Annuler : Abandonne les changements et ferme la fenêtre
DESACTIVATION D'UNE PERSONNE

Le nom de la personne apparait à l'écran pour vérification.

Après validation, la personne ne sera plus accessible dans la base.

La personne ne sera pas détruite physiquement de la base de données mais seulement désactivée logiquement.

  • Désactiver dans la base : Valide la désactivation.
  • Annuler : Abandonne et ferme la fenêtre
REACTIVATION D'UNE PERSONNE

Le nom de la personne apparait à l'écran pour vérification.

Après validation, la personne sera de nouveau accessible dans la base.

  • Réactiver dans la base : Valide la réactivation.
  • Annuler : Abandonne et ferme la fenêtre
