TEAM PASSWORD MODIFICATION

This interface aims to update team passwords.

A team is allowed to update it's own password.

Database administrator can update all passwords for all teams.

Team name appears for verification.

Old password must be entered if the user is not administrator.

The new password fiels is compulsary.

  • Save in Database : to valid modification
  • Cancel : to cancel modification and close the window
