Installation de CAPSIS sur votre poste

Nous vous conseillons de marquer cette page dans vos signets car elle contient des informations importantes

Pour utiliser CAPSIS, vous devez préalablement avoir Java Runtime Environment 6 sur votre poste. Il est possible de vérifier la version de Java sur votre poste en cliquant sur

  1. Démarrer
  2. Panneau de configuration
  3. Programmes
  4. Panneau de configuration Java
  5. Onglet “Java”
  6. Dans “Paramètres de Java Runtime Environment”, cliquez sur “Afficher”

Sous l'onglet “Utilisateur”, vous trouverez la version de Java Runtime Environment actuellement sur votre poste. Si vous ne trouvez pas le “Panneau de configuration Java”, il y a de fortes chances que Java ne soit pas encore installé sur votre poste.

Mise à jour ou installation de Java

Usagers au sein du MRNF

Si ce n'est pas déjà fait, demandez au Soutien à la clientèle (SAC-1@mrnf.gouv.qc.ca) d'installer CAPSIS sur votre poste de travail. Il s'agit d'une version antérieure de CAPSIS. Toutefois, lors de l'installation, on mettra votre version de Java à jour, de sorte que vous pourrez installer et utiliser la nouvelle version de CAPSIS contenant le module ARTEMIS.

Usagers à l'extérieur du MRNF

Téléchargez la dernière version de Java en cliquant ici. Vous n'avez qu'à suivre les instructions du site de Sun System.

Installation de CAPSIS

  1. Téléchargez l'utilitaire d'installation CAPSIS nommé “Artemisx.y.z-Succesx.y.z-setup.jar” sur votre bureau.
  2. Double-cliquez sur ce fichier
  3. Suivez les instructions.

ATTENTION: Il vous sera impossible d'installer CAPSIS dans un répertoire autre que votre répertoire personnel si votre système d'exploitation fonctionne sous Windows 7.

Utilisation de CAPSIS

CAPSIS dispose d'une aide en ligne. La plupart des menus comporte un bouton “Aide” qui vous dirigera à une page web contenant de l'information sur le menu en question. Vous pouvez également prendre connaissance du tutoriel de CAPSIS en cliquant ici.

Problèmes d'installation

CAPSIS ne démarre pas. Le problème le plus fréquent lors de l'installation de CAPSIS est le suivant: le répertoire contenant les fichiers Java n'a pas été ajouté au chemin de votre poste. Pour vérifier s'il s'agit bien de ce problème, recherchez tout d'abord le fichier “java.exe” en

  1. cliquant sur “Démarrer”
  2. cliquant sur “Rechercher”
  3. en inscrivant le nom “java.exe”.

Notez le répertoire où se trouve le fichier en question. Ensuite, démarrez un invite de commande en

  1. cliquant sur “Démarrer”
  2. cliquant sur “Commande” ou “Invite de commande”
  3. inscrivez la mot “path” dans la fenêtre qui se sera ouverte.

Vérifiez si le répertoire contenant le fichier “java.exe” fait bien partie de la liste des répertoires compris dans le chemin (“path”). Si ce n'est pas le cas, communiquez-nous le répertoire où se trouve le fichier “java.exe” et nous corrigerons la situation.

Pour ce problème ou pour tout autre, n'hésitez pas à nous joindre à l'adresse suivante : capsismrnfmodules@gmail.com.
