EDITION D'UN COMBUSTIBLE

Cette interface permet de créer, modifier ou visualiser des combustibles de type plante ou nappe dans la base de données.

CREATION/MODIFICATION

La saisie des données est organisée en 4 onglets :

  • ONGLET n°1 : L'équipe
  • ONGLET n°2 : La description du site
  • ONGLET n°3 : L'individu
  • ONGLET n°4 : Les commentaires

Les boutons suivant sont actifs en bas de l'écran.

Enregistrer dans la base : valide le nouveau combustible ou les modifications

Annuler : annule la saisie et ferme la fenêtre

* ONGLET n°1 : L'équipe

Les vérifications effectuées avant la validation sont :

Le choix de l'équipe est obligatoire

La date du relevé doit être au format : JJ/MM/AAAA

Le choix du premier opérateur de terrain est obligatoire

* ONGLET n°2 : Le site

Le choix du site est obligatoire

Editeur de sites : gestion des sites

* ONGLET n°3 : L'individu

Le choix de l'espèce est obligatoire

La latitude doit être un nombre compris entre 0 et 360 degrés

La longitude doit être un nombre compris entre 0 et 90 degrés

L'altitude doit être un nombre exprimé en mètres.

Les hauteurs et les diamètres doivent être des nombres exprimées en centimètres.

La hauteur de le diamètre max du houppier sont obligatoires.

La biomasse totale mesurée doit être un nombre exprimé en grammes.

La taux de couverture doit être un nombre compris entre 0 and 100 %.

* ONGLET n°4 : Commentaires

VISUALISATION

Les écrans sont identiques à ceux des modifications mais les zones de saisie et les boutons de validation ne sont pas accessibles.

Ceci arrive si :

  • vous n'êtes pas propriétaire de la données, donc vous n'avez aucun droit de modification
  • la plante est désactivée.
  • la plante est validée

Dans les 2 derniers cas, il faut d'abord réactiver ou devalider le combustible afin de pouvoir à nouveau le modifier.
