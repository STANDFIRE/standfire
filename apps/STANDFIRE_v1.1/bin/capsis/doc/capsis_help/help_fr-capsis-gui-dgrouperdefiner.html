Gestionnaire de Groupes

Principe

Créez des groupes (arbres, cellules) pour les suivre séparément dans les visualisateurs et extracteurs de données.

Onglet Groupe

Identification

  • Nouveau groupe : spécifiez son nom
  • Groupe existant : modifiez son nom si nécessaire

Type d'éléments

  • Choisissez le type des éléments du groupe (arbres, cellules…)

Type de groupe

  • Un groupe statique ne retient que les identifiants des éléments sélectionnés pendant la création du groupe
  • Un groupe dynamique retient les règles définies pendant la création du groupe et les ré-applique à chaque utilisation

Onglet Définition

Utilisez le gestionnaire de filtres pour construire votre groupe.
