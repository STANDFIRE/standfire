Module PP3

Volume de résidus de coupe (+ billons récoltés) / Origine

Pour faire fonctionner ce visualiseur, vous devez activer le calcul du bois mort lors de l'initialiation du projet en cliquant sur le bouton “Bois Mort”.

Pour que des données s'affichent, il faut se trouver sur une étape d'éclaircie.

On choisit les catégories de bois à afficher en cliquant sur le bouton droit pour accéder à la configuration du graphique puis en choisissant l'onglet “Commun”.
