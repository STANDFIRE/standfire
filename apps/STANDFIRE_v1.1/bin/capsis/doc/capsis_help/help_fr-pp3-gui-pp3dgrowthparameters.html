MODULE PP3

Critères d'évolution

Cette boîte de dialogue permet de dialoguer avec le modèle de croissance.
On y acède par le menu contextuel (sur l'étape clic bouton droit de la souris) -> Evolution
 

Choix des critères d'évolution 

  • 1. Zone Critères d'évolution

    • Choix obligatoire entre l'âge et/ou la Hdom et/ou la Cdom et/ou la Cg à atteindre.
    L'évolution se termine lorsque le premier de ces critères est atteint ou dépassé.

  • 2. Zone Valeurs Seuils (facultative)

    • L'évolution est stoppée si une de ces valeurs seuils est dépassée à la baisse (pour les limites basses) ou à la hausse (limite haute).
    Pour continuer l'évolution, l'utilisateur doit manuellement entrer une nouvelle valeur (ou supprimer la précédente).


    Tous les visualiseurs synchronisés sur l'étape de départ montrent la dernière étape calculée (voir Sélecteur).
