Marteloscope

A tool to cut trees which identifiers are read in a file prepared in a spreadsheet.

Option 1

The file contains a simple list of identifiers (one per line without considering the first line which is ignored).

Sigisbert Dupondt
19
26
35
34

Option 2

Load a csv file (fields separator: ”;”) with this format:

Arbres;Equipe 1;Sigisbert Dupondt;Equ. 3;EQUIPE QUATRE
21;;;;
11;x;;;
19;;m;moche;1
17;;;;
26;;M;fourchu;1

Through the user interface, it is then possible to choose the team which column will be considered. All the trees with some text in this column will be cut (non empty column).

Note: For both options, lines starting with a ”#” will be ignored (considered as comment lines).
