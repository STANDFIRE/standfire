DESACTIVATION OU REACTIVATION D'UNE FORME

Cette interface permet de désactiver ou réactiver une forme de type plante, nappe ou échantillon dans la base de données.

Ces actions ne sont possibles que sur les formes dont l'utilisateur est le propriétaire.

DESACTIVATION

Les données relatives à la forme apparaissent à l'écran pour vérification.

Après validation, la forme ne sera plus accessibles dans la base pour la création de scène virtuelle ou pour l'exportation dans des modèles comme Firetec.

La forme ne sera pas détruite physiquement de la base de données mais seulement désactivée logiquement.

  • Enregistrer dans la base : Valide la désactivation.
  • Annuler : Abandonne et ferme la fenêtre
REACTIVATION

Les données relatives à la forme apparaissent à l'écran pour vérification.

Après validation, la forme sera de nouveau accessible dans la base.

  • Enregistrer dans la base : Valide la réactivation.
  • Annuler : Abandonne et ferme la fenêtre
