twoe

The twoe software includes both an R package for estimating tree species demographic model parameters and a Capsis module for simulating forest dynamics. The R package includes statistical functions to estimate the parameters of the growth, mortality and recruitment models for each tree species individually. Parameter inference is done in a hierarchical Bayesian framework to manage the low number of observations associated to rare species, measurement errors and variable time interval between censuses. Estimated species parameters can be directly used to simulate forest dynamics using the twoe model in Capsis. The natural forest dynamics can be forecasted as well as a the dynamics following different intensity of logging. Capsis provides tools to look at the conservation of several forest ecosystem services such as wood production, biodiversity and carbon stock.

For the twoe R package please visit http://twoe.org.
