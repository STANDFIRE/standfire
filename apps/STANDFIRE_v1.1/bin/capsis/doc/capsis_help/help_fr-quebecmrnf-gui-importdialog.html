Utilitaire d'importation de fichiers de données

Ce menu vous permet de lire et d'importer les données des fichiers qui sont en format ”.dbf” ou ”.csv”.

Sélectionnez dans le menu déroulant de droite les champs qui correspondent aux champs obligatoire mentionnés à la gauche du menu. Notez que certains champs sont facultatifs. Si votre fichier ne comprend pas ces champs, vous n'avez pas à les associer à ceux du menu déroulant . Par ailleurs, l'utilitaire d'importation reconnaît que votre fichier d'entrée ne comprend qu'une seule strate ou un seul peuplement lorsqu'aucun champ n'est associé à l'identifiant de strate.

Pour associer un champ de votre fichier ”.dbf” ou ”.csv” à un champ obligatoire, vous n'avez (1) qu'à cliquer sur un champ dans le menu déroulant. Cliquez (2) ensuite sur la flèche qui se trouve à la droite du champ obligatoire auquel il correspond. Le nom du champ sélectionné dans le menu déroulant apparaîtra à la droite du champ obligatoire.

ASTUCE: La description d'un champ apparaît dans une fenêtre lorsque vous passez le curseur de la souris au-dessus de ce champ.

Validation des données d'entrée

Artemis applique certains filtres pour s'assurer que vos données d'entrée correspondent à celles qui ont servi à étalonner le modèle:

  • La latitude doit être comprise entre 45 et 53 degrés.
  • La longitude doit être comprise entre -80 et -58 degrés (attention les valeurs doivent être négatives).
  • L'altitude doit être comprise entre 0 et 1300 m.
  • Les précipitations annuelles moyennes si elles sont disponibles doivent être comprises entre 0 et 1700 mm.
  • La température annuelle moyenne si elle est disponible doit être comprise entre -7 et +10 degrés Celsius.

IMPORTANT: Si vos données ne répondent pas à ces critères, vous verrez apparaître un message d'erreur.

OPTIONS: Enregistrer sous/Ouvrir

L'option “Enregistrer sous” vous permet d'enregistrer les associations entre les champs de votre fichier de données et ceux nécessaires à la simulation. Si vous utilisez le même fichier ou un fichier de structure semblable lors d'une session ultérieure, vous n'avez qu'à cliquer sur le bouton “Ouvrir” pour relire ces associations de champs.
