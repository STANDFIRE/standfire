Module Sapin

Auteur

François de Coligny (INRA), UMR AMAP, botAnique et bioinforMatique de l'Architecture des Plantes, TA 40 / PS2 - Boulevard de la Lironde, 34398 Montpellier Cedex 5

Objectifs

Sapin est un module d'entrainement qui est utilisé pendant les formations Capsis.

Description

Sapin est un module d'enseignement. C'est un modèle arbre dépendant des distances. Les propriétés de l'arbre sont : id, x, y, z, d1.3m (cm), height (m), age, heightIncrement, dbhIncrement, crownBaseHeight, crownRadius.

L'évolution est calculée pour chaque arbre : diamètre et hauteur. Pendant la formation, une méthode de régénération stochastique simple est ajoutée.

Les fichiers inventaires contiennent une ligne par arbre.
