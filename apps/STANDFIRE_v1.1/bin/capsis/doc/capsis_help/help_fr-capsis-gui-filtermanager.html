Gestionnaire de filtres

Principe

Sélectionnez des éléments (ex: arbres, cellules de terrain, individus…) en combinant des filtres compatibles avec le module associé au projet. Les filtres opèrent par raffinements successifs.

Filtres Disponibles

Sélectionnez un filtre dans la liste des filtres disponibles pour faire apparaître sa description dans la zone de texte située en bas du gestionnaire. Pour utiliser un filtre, désignez le avec la souris et faites le passer dans la liste des filtres sélectionnés par le bouton .

Filtres sélectionnés

Les filtres sélectionnés sont paramétrables par un panneau de configuration qui apparaît sur la partie droite du gestionnaire.

Le résultat du paramétrage peut être contrôlé dans un visualiseur témoin, choisi dans une liste déroulante de témoins compatibles possédant leur propre paramétrage.

Quand la sélection change dans le panneau de configuration du filtre, il est possible de raffraichir le témoin par action sur le bouton .

Si plusieurs filtres sont sélectionnés, chacun opère par raffinement du résultat de la sélection de son prédécesseur dans la liste. Seul le dernier filtre sélectionné est paramétrable. Pour modifier le paramétrage d'un filtre précédent, supprimez d'abord tous ses successeurs a l'aide du bouton .
