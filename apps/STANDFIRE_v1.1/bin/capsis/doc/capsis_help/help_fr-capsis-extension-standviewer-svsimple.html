Visualiseur simple

Principe

Visualisez le peuplement spatialisé associé à une étape de projet sur une carte.

Mode d'emploi

Chaque arbre est représenté par un point dessiné à sa position réelle dans le peuplement. Un libellé peut montrer son identifiant numérique (icône préférences).

Information sur un ou plusieurs arbres

Choisissez le mode de sélection dans la liste proposée, puis sélectionnez le ou les arbre(s) :

  • Pour sélectionner un arbre, double-cliquez dessus avec le bouton gauche.
  • Pour sélectionner un groupe d'arbres : définissez un rectangle de sélection englobant avec le bouton droit.

Zoom

  • Zoom avant : définissez un rectangle de zoom englobant la zone d'intérêt avec le bouton gauche.
  • Zoom arrière : cliquez sur l'image avec le bouton droit.

Déplacement sur l'image

  • Tracez un vecteur de translation sur l'image en maintenant la touche Control enfoncée.
