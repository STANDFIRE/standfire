Selection of trees by thresholds

Select some trees by specifying low / high thresholds on their age, diameter or height:

  • choose Age, Diameter or Height,
  • enter a low threshold for the considered property,
  • enter a high threshold.

The 'Available values' button sets the min and max available values for the current property.

Both thresholds are optional: if not specified, the low threshold is considered equal to zero and the high one is set to the maximum possible value to have no effect. It is thus possible to specify “more than value a”, “less than value b” or “between value a and value b”.
