SHAPE CREATION FOR A MEASURED LAYER

A measured layer has been sampled in field with the CUBE method.

The layer is composed of a CORE shape and/or an EDGE shape.

These shapes are composed of T (Top) C (Center) or B (Bottom) cubes, each type having the same perticles list and biomasses.

Fill here shapes dimensions in cm :

  • voxels dimensions
  • CORE sample height (optionnal).
  • EDGE sample height (optionnal)
  • CORE shape height (Z) and diameter (X) (optionnal).
  • EDGE shape height (Z) and diameter (X) (optionnal).
  • Max spead width of the layer.
SAMPLES

When a sample exists, it is no more possible to modify voxel sizes.

At least one sample is compulsary (CORE ou EDGE)

It is impossible to have several CORE sample or saveral EDGE samples for the same layer, except if some of them are desactivated.

2D SHAPES

The 2D shape representing the layer, is composed of a CORE part and/or an EDGE part.

The CORE sample will be used in priority to build the CORE shape part.

The EDGE sample will be used in priority to build the EDGE shape part.

If there only one sample, it will be used to build the both part of the shape.

It is impossible to have sevaral 2D shape for the same layer, except if some of them are desactivated.
