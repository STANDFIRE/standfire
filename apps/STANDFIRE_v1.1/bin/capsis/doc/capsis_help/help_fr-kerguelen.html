Objectifs

Le module Kerguelen est dédié à la question de la colonisation des Iles Kerguelen par les salmonidés. Il est dérivé du module Bidasoa, avec un cycle de vie simplifié, mais intègre désormais un environnement marin. L'objectif principal est de générer des simulations permettant de discriminer les différentes hypothèses expliquant les patrons de colonisation observés: densité-dépendance, dispersion en stepping-stone, avantage sélectif des formes migrantes, etc.

Ce module est spatialisé, utilise une échelle individuelle, et profite de l'extension Genetics 2.0 de Capsis-4.

Le fichier d'inventaire contient des données d'espèce, de carte génétique, d'individus, de tronçons, seuils et environnement marin.

Ce module est en construction, une documentation détaillée sera fournie ultérieurement.
