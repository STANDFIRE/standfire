LISTE DE FORMES POUR UN COMBUSTIBLE

Tous les formes rattachées à un même combustible et stockées dans la base de données apparaissent dans cette liste.

Elle contient le type de forme (plante, nappe ou échantillon), l'information XZ, XZ_XY ou XYZ indiquant si la forme est en 2D, 2 fois 2D ou 3D, la méthode d'échantillonnage utilisée (cube ou cage) les dimensions du houppier et la mention si celle-ci est validée ou désactivée.

En haut de l'écran à gauche, un cadre rappelle les informations concernant le combustible.

Les actions possibles sont:

  • Ajouter: pour ajouter une nouvelle forme à cette plante
  • Modifier: pour modifier une forme existante
  • Valider: pour valider une forme
  • Activer/Désactiver: pour désactiver ou réactiver une forme
  • Fermer: pour fermer la fenêtre

La fonction valider permet d'indiquer que la saisie de cette forme est terminée.

La fonction désactiver permet d'interdire l'utilisation d'une forme sans pour autant en détruire physiquement les données.
