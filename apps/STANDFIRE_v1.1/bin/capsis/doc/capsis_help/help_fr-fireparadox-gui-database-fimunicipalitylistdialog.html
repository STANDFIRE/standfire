EDITION DES MUNICIPALITES

Toutes les municipalités répertoriées dans la base de données apparaissent dans cette liste.

Elle contient le code du pays, le nom de la municipalité et la mention désactivée si la municipalité n'est plus active.

Des sélecteurs en haut de la liste permettent de réduire la liste à un code pays particulier.

Les actions possibles sont:

  • Modifier: pour modifier une municipalités existante
  • Ajouter: pour ajouter une nouvelle municipalités
  • Désactiver/Réactiver: pour désactiver ou réactiver une municipalités
  • Fermer: pour fermer la fenêtre

La fonction désactiver permet d'interdire l'utilisation d'une municipalité sans pour autant détruire les données lui faisant déjà référence.
