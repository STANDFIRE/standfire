EDITION DES SITES

Cette interface permet de créer, modifier, désactiver ou réactiver des sites dans la base de données.

CREATION D'UN SITE

Saisir le code du nouveau site ainsi que toutes les données nécessaires.

Les vérifications avant validation sont:

Le code du site est obligatoire

Le nom de la municipalité est obligatoire

La latitude doit être un nombre compris entre 0 et 360 degrés

La longitude doit être un nombre compris entre 0 et 90 degrés

L'altitude doit être un nombre.

La pente doit être un nombre compris entre 0 et 360 degrés

L'orientation doit être un nombre compris entre 0 et 360 degrés

  • Editeur de municipalités: gestion des municipalités
  • Gérer les événements : permet de gérer la liste des événements liés à ce site
  • Enregistrer dans la base : valide le nouveau site et ses événements
  • Annuler : annule la saisie et fermer la fenêtre
MODIFICATION D'UN SITE

Saisir le nouveau code du site ainsi que toutes les modifications nécessaires.

  • Editeur de municipalités: gestion des municipalités
  • Gérer les événements : permet de gérer la liste des événements liés à ce site
  • Enregistrer dans la base : valide les modifications
  • Annuler : annule les modifications et fermer la fenêtre
DESACTIVATION D'UN SITE

Le code du site apparait à l'écran pour vérification.

Après validation, le site ne sera plus accessible dans la base.

Le site ne sera pas détruit physiquement de la base de données mais seulement désactivé logiquement.

Toutes les données lui faisant référence (plantes, nappes, échantillons) seront conservées et pourront être utilisées dans le Fuel manager.

  • Désactiver dans la base : Valide la désactivation.
  • Annuler : Abandonne et ferme la fenêtre
REACTIVATION D'UN SITE

Le code du site apparait à l'écran pour vérification.

Après validation, le site sera de nouveau accessible dans la base.

  • Réactiver dans la base : Valide la réactivation.
  • Annuler : Abandonne et ferme la fenêtre
