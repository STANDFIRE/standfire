Melies thinner

This intervener allows you to setup target basal area or RDI for each species. The thinning coefficient Kg is applied on the last computed value. Ie : if you set RDI, G is calculated. Changing Kg recompute RDI.

Detail of the formulae used :

  • Kg = ( thinned_trees_Dg / initial_Dg)² = ( initial_N * thinned_G ) / (thinned_N* initial_G)
  • RDI = stems_per_ha * exp( -13.06 + 1.83 * ln(initial_Dg) )

For further reading see : Vallet et Pérot 2011, Journal of Vegetation Science, 22(5):932–942)
