Oakpine1

Modèle pour les peuplements mélangés chêne sessile (Quercus petraea) pin sylvestre (Pinus sylvestris)

Authors:

Thomas Perot, Marie Ange Ngo Bieng, François Goreaud

Description:

Oakpine1 est un modèle individu centré spatialement explicite. La compétition est décrite à l'aide d'indices de compétition locaux. Le modèle de croissance est relié à des modèles de structure spatiale basés sur les processus ponctuels. Si la position des arbres n'est pas disponible, les modèles de structure spatiale peuvent être lancés lors de la phase d'initialisation du module de façon à générer une carte du peuplement.
