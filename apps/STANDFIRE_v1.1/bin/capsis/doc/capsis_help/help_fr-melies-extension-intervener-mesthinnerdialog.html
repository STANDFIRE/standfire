Eclaircisseur Melies

Cet outil d'intervention permet de spécifier un objectif en termes de RDI ou de surface terrière pour chaque essence. Le coefficient d'éclaircie Kg est appliqué sur la valeur qui a été recalculée. Ie : si vous fixez le RDI, G est calculé. Changer Kg recalcule G

Détail des formules utilisées :

  • Kg = ( Dg_arbres_eclaircis / Dg_initial)²
  • Kg = ( N_initial * G_arbres_eclaircis) / ( N_arbres_eclaircis * G_initial )
  • RDI = tiges_par_ha * exp( -13.06 + 1.83 * ln(Dg_initial) )

Pour plus de détails, voir : Vallet et Pérot 2011, Journal of Vegetation Science, 22(5):932–942)
