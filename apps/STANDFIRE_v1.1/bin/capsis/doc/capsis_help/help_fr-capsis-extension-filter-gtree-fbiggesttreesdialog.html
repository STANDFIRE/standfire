Sélection des n plus gros arbres

Principe

Sélectionnez les n plus gros arbres selon leurs diamètre à 1.3m.

Saisissez le nombre d'arbres souhaité, un entier supérieur à 1.

Le bouton Valeur possible indique le nombre d'arbres maximum que l'on peut choisir.

Remarque: le filtre trie les arbres sur deux critères : diamètre décroissant, puis identifiant décroissant pour répondre de manière consistante. Ainsi, si plusieurs arbres ont le même diamètre, on tirera toujours les même n plus gros.
