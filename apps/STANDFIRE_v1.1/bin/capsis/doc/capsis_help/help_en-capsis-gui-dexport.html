Export

Principle

Choose a format and a file name to export data based on the current step.

Method

The available formats are proposed in a combo box. Selecting one format shows information about it.

The target file will contain the exported data. Choose a localization on the disk and a name for the file.

When all is correct, hitting ok will export the data in the given file.

Note: the various formats can generate:

  • a description of the scene under the current step,
  • a description concerning several steps of the scenario defined from the root step to the current step.

In some cases, the export feature can ask for parameters in a dialog box, or sometimes create several files with names based on the given name, augmented with suffixes.
