Inspecteur

Principe

Ce composant dynamique vous permet d'examiner les propriétés des objets sur lesquel il est positionné.

Visualisation

L'inspecteur est ouvert sur un objet cible. Les propriétés de l'objet sont listées dans une table, avec leur valeur en regard.

Certaines propriétés sont des références d'autres objets : double-cliquez dessus avec le bouton gauche pour que l'objet référencé remplace la cible actuelle.

Navigation

A tout moment, il est possible d'utiliser les boutons de la barre de navigation :

  • Cible précédente :
  • Cible suivante :
  • Cible de départ :

La barre de navigation contient également le type de l'objet courant (nom de classe java) ainsi qu'un bouton pour ouvrir une nouvelle vue de l'objet. Si plusieurs choix sont possibles, une liste est proposée.

Quelles propriétés

Les propriétés montrées sont :

* les variables de l'objet cible (plus exactement les variables d'instances publiques) * le résultat de l'invocation dynamique de ses accesseurs publics (méthodes de la forme getNomPropriété () ). Dans ce dernier cas, le préfixe “get” est supprimé pour former le nom de la propriété (ex: getHeight () → Height)

Les propriétés privées ou protégées des objets ne sont pas montrées.
