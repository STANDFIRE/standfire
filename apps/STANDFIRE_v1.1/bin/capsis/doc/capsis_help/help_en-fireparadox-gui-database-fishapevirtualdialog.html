SHAPE CREATION FOR A VIRTUAL PLANT

Fill the following options :

  • Sample creation : YES/NO
  • Voxels dimensions in cm : height (Z), main diameter (X), perpendicular diameter (Y)
  • Sample height (Z)only if sample creation is selected
  • Shape dimensions in cm : height (Z), main diameter (X), perpendicular diameter (Y)
SAMPLE CREATION OPTION

First you will have to build a “virtual” sample composed of a cube column. Each cube can be T (Top) C (Center) B (Bottom) type, each type having a particle list and measured biomasses.

This sample is NOT from field measurement, it is only a way to help user to build 3D shape more quickly.

Then the 3D shape will be build with sample cubes.

NO SAMPLE CREATION

The 3D shape will be directly build with cubes, all diffferents.
