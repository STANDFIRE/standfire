Définir un scénario avec éclaircies automatiques:
  • Indiquer l'âge final et/ou le diamètre dominant final. Si les 2 champs sont remplis, le scénario s'arrêtera lorsque la première condition sera remplie.
  • Indiquer le Rdi initial et le Rdi final. Une première éclaircie sera pratiquée immédiatement si le Rdi initial est inférieur à la valeur courante affichée. Des éclaircies seront ensuite déclenchées dès que le Rdi courant dépassera le rdi objectif défini par la droite [âge initial, Rdi initial] - [âge final, Rdi final] et que les conditions définies dans le cadre “Intensité des éclaircies” seront remplies.

Intensité des éclaircies

Le domaine de variation autour du rdi cible indique la tolérance admise autour du rdi objectif pour les éclaircies automatiques :

  • l'éclaircie sera déclenchée dès que le rdi courant dépassera la valeur “rdi objectif + domaine de variation” ;
  • l'éclaircie ramènera le rdi à la valeur “rdi objectif - domaine de variation”.

Les éclaircies automatiques ne seront pas déclenchées avant que le nombre d'années minimal requis soit atteint.

Type d'éclaircie

Le type d'éclaircie définit la façon dont seront sélectionnés les arbres à éclaircir pour atteindre le Rdi objectif :

  • dans une éclaircie standard, un diamètre dSup est défini (diamètre moyen que l'on obtiendrait en ne coupant que les petits arbres pour obtenir le Rdi objectif) au delà duquel aucun arbre ne sera coupé. Les petits arbres auront une probabilité d'autant plus forte d'être éclaircis que leur diamètre est inférieur à dSup ;
  • dans une éclaircie par le haut tous les arbres auront la même probabilité d'être éclaircis.
