Intervention basée sur le Rdi

Paramètres de l'éclaircie

Indiquer le Rdi objectif à atteindre après l'éclaircie.

Le type d'éclaircie définit la façon dont seront sélectionnés les arbres à éclaircir pour atteindre le Rdi objectif :

  • dans une éclaircie standard, un diamètre dSup est défini (diamètre moyen que l'on obtiendrait en ne coupant que les petits arbres pour obtenir le Rdi objectif) au delà duquel aucun arbre ne sera coupé. Les petits arbres auront une probabilité d'autant plus forte d'être éclaircis que leur diamètre est inférieur à dSup ;
  • dans une éclaircie par le haut tous les arbres auront la même probabilité d'être éclaircis.
