AJOUT D'UNE FAMILLE DE PARTICULE

Cet interface permet de choisir une nouvelle famille de particule à ajouter dans un échantillon.

Une fois cette opération effectuée, des biomasses pourront être saisies dans l'échantillon, et au niveau plante, des paramètres de type MVR, SVR etc… pourront être saisis.

Choisir la nouvelle famille de particule dans la liste et cliquer sur Ajouter

Pour fermer la fenêtre sans ajouter de nouvelle particule, cliquer sur Annuler
