MODULE PP3

Initialisation du projet

Cette boîte de dialogue permet d'initialiser un projet :
  1. Choix du peuplement qui constituera l'étape racine
  2. Vérification du paramétrage du module PP3
 

Choix du peuplement de départ

La première partie de cette boîte de dialogue permet d'initialiser le module pour la simulation qui commence.

Cette initialisation consiste à choisir le type de peuplement qui constituera l'étape racine.

On peut choisir entre le chargement d'un fichier d'inventaire, la création d'un peuplement fictif et le choix d'un peuplement réel sélectionné selon différents critères.

  1. Chargement d'un fichier d'inventaire

Les fichiers d'inventaires doivent être positionnés dans le répertoire C:\capsis4\data\PP3
Un exemple de fichier d'inventaire est fourni dans le fichier : model_input.pp3 localisé dans data\PP3 du répertoire d'installation de capsis.
Pour plus de détails, voir la description du module PP3.

  1. Générer un peuplement fictif

L'utilisateur peut initialiser une simulation à partir d'un peuplement fictif  âgé de 12 ans sur la base de son indice de fertilité et de sa densité.

  1. Choix d'un peuplement dans une liste de critères

Des fichiers d'inventaire réels sont proposées et accessibles après sélection de critères. (Option non activée)

Paramétrage du module

Cette boîte de dialogue permet de vérifier le paramétrage du module pour la simulation qui commence.

Ces paramètres seront gardés pour toute la durée de la simulation pour ce projet.
