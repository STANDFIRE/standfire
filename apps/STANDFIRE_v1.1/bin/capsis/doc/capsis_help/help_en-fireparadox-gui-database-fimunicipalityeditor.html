MUNICIPALITIES EDITION

This interface aims at create, update, desactivate ou reactivate munucipalities in the database.

MUNICIPALITY CREATION

Enter the new municipality name and country. These two fields are compulsory.

  • Save in Database : to valid the new municipality
  • Cancel : to cancel and close the window
MUNICIPALITY MODIFICATION

Enter the municipality new name if country. These two fields are compulsory.

  • Save in Database : to valid modifications
  • Cancel : to cancel modifications and close the window
MUNICIPALITY DESACTIVATION

The municipality name appears on the screen for verification.

After validation, the municipality will no longer be available in the database.

  • Cancel : to cancel and close the window
MUNICIPALITY REACTIVATION

The municipality name appears on the screen for verification.

After validation, the municipality will be again available in the database.

  • Reactivation : to validate the reactivation.
  • Cancel : to cancel and close the window
