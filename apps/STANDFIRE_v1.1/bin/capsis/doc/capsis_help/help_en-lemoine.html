Module Lemoine

A completed description is given in French.

1. Authors

Bernard Lemoine INRA - Unité de recherches forestières de Pierroton

Céline Meredieu INRA UMR BIOGECO Site Forêt - Bois 69 route d'Arcachon 33 612 CESTAS Cedex

2. Objectives

Lemoine is a forest growth model designed to the stand dynamics and silviculture interactions. It can be used to compare thinning strategies.

3. Description

Lemoine module is an Empirical Stand Model.

3.1 Inputs

The user has to give site fertility, initial age, density and also dominant and mean girths.

3.2 Outputs

Various curves, histograms and scatterplots of tree and stand dendrometric characteristics.

4. Validity

Lemoine model was calibrated for Maritime pine stands located in the coastal area of the Landes de Gascogne, France.

5. Papers

Lemoine B., 1996 - MODELE DE CROISSANCE POUR LES FORETS DE DUNES - CHOIX DU MODELE - PROPOSITION POUR UNE VALIDATION, Bordeaux (France), Document interne, 30p.
