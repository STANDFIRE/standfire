Intervention

Principe

Choisissez une méthode d'intervention à appliquer sur l'étape de référence.

Marche à suivre

Les méthodes d'intervention créent une nouvelle étape à la suite de l'étape de référence, portant le peuplement après intervention.

Il est possible de restreindre l'intervention à un groupe d'individus. Cochez la case groupe et choisissez le groupe dans la liste. Si certaines méthodes d'intervention ne sont pas compatibles avec l'option groupe, elles disparaissent de la liste.

Les méthodes d'intervention sont regroupées par type. Choisissez d'abord le type de l'intervention : les méthodes correspondantes apparaissent dans la liste de sélection.

Quand une méthode est sélectionnée, sa description est proposée. Pour ouvrir l'outil sélectionné, appuyez sur OK.
