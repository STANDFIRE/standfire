Objectives

The AMAPsim module is used for connexion between some Capsis modules (PP3 - Maritime pine…) and the AMAPsim growth simulator. This module loads inventories and builds Capsis projects from data returned by AMAPsim.

Description

This modules provides no evolution method. For more details, see the documentation 'Connexion Capsis AMAPsim' (french) on the Capsis web site.

Outputs

The various viewers and graphs in Capsis can be used to analyse the properties of the stands in the AMAPsim module. Capsis export formats can also be used for studies in other software.
