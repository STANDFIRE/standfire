Graphical trees selection

Select the trees directly with the mouse. The selected trees appear in red.

Customizing the display

Change the number of displayed labels by changing the value and validate. Change the magnify factor for the trees diameter in the same manner. Swap trees identifiers and trees diameters at 1.3 m with the diameter check box.

Selection / Deselection

To (de)select a tree, double click on it with the left button To (de)select a group of trees: define a selection rectangle with the right button

To deselect all, use the red button

Information about one or more trees

Select the tree(s) while holding down the Control key.

Zoom

Zoom in : define a zoom rectangle with the left button Zoom out: click on the image with the right button

Move on the image

Draw a translation vector on the image while holding down the Control key.
