Etapes visibles

Principe

Choisissez les étapes qui seront visibles dans le gestionnaire de projets.

Démarche

Toutes les étapes, intermédiaires comprises, sont montrées dans cette boîte de dialogue.

Cliquez sur les boutons étapes pour changer leur statut. Les boutons rouges sont les boutons visibles qui apparaîtront dans le gestionnaire de projets.

Note : il est impossible de rendre invisible certaines étapes :

  • L'étape racine
  • Les étapes finales
  • Les étapes ayant plusieurs filles (noeuds)
