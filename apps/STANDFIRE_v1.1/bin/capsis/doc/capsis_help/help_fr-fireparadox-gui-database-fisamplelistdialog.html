LISTE DES ECHANTILLONS

Tous les objets combustibles de type échantillons stockés dans la base de données apparaissent dans cette liste.

Elle affiche les codes équipe, site, le nom de l'espèce, son type (unique, coeur, bord), la méthode utilisée pour l'échantillonage (cube, cage), ses dimensions, la référence de la plante ou nappe, l'origine de l'échantillon (virtuel ou mesuré) et la mention si celui-ci est désactivé.

La mention validée concerne la plante ou la nappe de rattachement.

Les sélecteurs du haut permettent de restreindre la taille de cette liste.

Les actions possibles sont:

  • Editer: pour editer l'échantillon
  • Désactiver/Réactiver: pour désactiver ou réactiver l'échantillon
  • Fermer: pour fermer la fenêtre

La fonction Editer permet de voir les données de cet échantillon. Les modifications sont possibles si l'utilisateur est propriétaire de la donnée et que l'échantillon n'est pas désactivé. Dans le cas contraire, l'échantillon devra déjà être ré-activé pour être à nouveau modifiable. Il en est de même si la plante ou la nappe à laquelle l'échantillon est rattaché, est déjà validée.

La fonction désactiver permet d'interdire l'utilisation d'un échantillon sans pour autant détruire les données lui faisant déjà référence.
