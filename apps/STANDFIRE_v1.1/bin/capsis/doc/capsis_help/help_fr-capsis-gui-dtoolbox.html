Boîte à outils

Principe

Choisissez un outil à appliquer sur l'étape de référence.

Marche à suivre

La liste contient les noms des outils réputés compatibles avec le module associé à l'étape de référence.

Quand un outil est sélectionné dans la liste, sa description est proposée. Pour ouvrir l'outil sélectionné, appuyez sur OK.
