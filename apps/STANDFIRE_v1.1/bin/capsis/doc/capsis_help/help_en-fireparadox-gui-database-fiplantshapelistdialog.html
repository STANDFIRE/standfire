FUEL SHAPE LIST

All shapes attached to a fuel and stored in the database appear in this list.

It contains for each the type (plant, layer or sample), XZ, XZ_XY ou XYZ means that the shape is 2D, 2 X 2D or 3D, the sampling méthod (cubes or cage), crown dimensions and the mention if the shape is validated or désactivated.

At the top od the screen, on the left, a panel resume fuel general informations.

Control buttons are:

  • Add: to add a new shape
  • Edit: to edit a shape
  • Desactivate/Reactivate: to desactivate or reactivate a shape
  • Close: to close the window

Add action is not available if the user is not the owner of the plant, or if the plant is desactivated or validated. It is also impossible to enter several active 3D shapes for a plant or several active 2D shapes for a layer.

Edit action is usefull to see the shape. Modifications are possible if the user is not the owner of the plant and if this plant is neither desactivated or validated.

Desactivate action is usefull to denied utilisation of a shape without deleting data physically in the database. This action is not available if the user is not the owner of the plant, or if the plant is desactivated or validated.

Measured plants

With cube sampling method, shape list can present at least, one SAMPLE UNIQUE and one 3D shape. 2D and 2 * 2D shapes can also appears, to help user to build automatically 3D shapes.

With cage sampling method, shape list presents only one SAMPLE UNIQUE and one 3D shape. It is not possible to have 2D shapes.

Virtual plants

For virtual plants, SAMPLES creation is optionnal. If a virtual SAMPLE is created, it is possible only with cube sampling method

Measured layers

Cage sampling method is not available for layers.

Shape list can present 2 types of samples CORE and/or EDGE and one 2D shape composed with a core part and/or an edge part.

Virtual layers

For virtual layers, SAMPLES creation is optionnal.
