Sélection d'arbres par cellules

Sélectionnez des arbres en désignant les cellules de terrain qui les contiennent.

Modification de l'affichage

Changez le nombre de labels affichés et modifiant la valeur et en validant. Modifiez le facteur de grossissement du diamètre des arbres de la même manière. Permutez les identifiants par les diamètres des arbres (d1.3m) en agissant sur la case à cocher Diamètres. Modifiez le niveau auquel la sélection se fera dans le cas de plusieurs niveaux de cellules imbriqués.

Sélection / désélection des cellules

Pour (dé)sélectionner une cellule, double-cliquez dessus avec le bouton gauche. Pour (dé)sélectionner un groupe de cellules : définissez un rectangle de sélection englobant avec le bouton droit.

Pour tout désélectionner, appuyez sur le bouton rouge.

Information sur un ou plusieurs arbres

Sélectionnez le ou les arbre(s) en maintenant la touche Control enfoncée.

Zoom

Zoom avant : définissez un rectangle de zoom englobant la zone d'intérêt avec le bouton gauche. Zoom arrière : cliquez sur l'image avec le bouton droit.

Déplacement sur l'image

Tracez un vecteur de translation sur l'image en maintenant la touche Control enfoncée.
