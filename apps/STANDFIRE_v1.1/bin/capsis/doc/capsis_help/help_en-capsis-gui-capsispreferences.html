Capsis Preferences

On startup

Choose if you would like to see the “Welcome to Capsis” dialog.

Tune the help file download frequency in days.

User Interface

Change the display mode of the project manager to have a more compact view.

Activate the borders arround the buttons in the Capsis toolbar.

Tune the thickness of the border of the internal frames containing the graphics (not compatible with all Look and Feels).

Change the color of selection in the project manager.

External Tools

Choose the Web browser to show the documentation pages.

Directories

Check or change the paths of the main Capsis directories.

Look And Feel

To change the appearance of Capsis, choose a “Look and Feel” in the available list and select it.
